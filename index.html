Introduction.

Cette page regroupe les notes de cours et les ressources nécessaires au module Java EE / Spring. Le plan de cours est le suivant :

  • Java 8, Maven, Eclipse
  • Servlet et JSP
  • War et Contexte Web
  • Persistance : de JDBC à JPA
  • Spring : contexte et web-mvc
  • Spring-data et gestion transactionnelle
  • Spring-boot, spring-loaded
  • JMS
  • Pour aller plus loin : MongoDB, ElasticSearch
  • Philosophie micro-services
