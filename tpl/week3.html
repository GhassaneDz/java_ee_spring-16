Suivez les consignes de la partie week301

Vous utiliserez le check.jar fourni dans le projet git, pensez à le mettre à jour avec git pull.
Installer ce jar dans votre repository local.
Ajouter cet artefact comme dépendance dans votre pom, en scope test

Tester votre programme.
Exécuter la classe boot.Main avec les dépedances de test :
mvn -q exec:java -Dexec.mainClass=boot.Main -Dexec.classpathScope=test -Dexec.arguments=week301
ajoutez l'option -DNoProxy si vous n'êtes pas sur le réseau de l'université.
